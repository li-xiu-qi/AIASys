# AIASys 平台指南

AIASys 工作平台的完整使用指南。覆盖核心概念、Agent 工具目录、AutoTask 配合模式、UV 运行环境、Docker 沙盒资源、环境变量、多维数据表和 Skill 协作方式。

## 适用场景

- 需要了解 AIASys 平台有哪些工具和能力
- 配合 AutoTask 执行后台任务
- 管理环境变量、数据表或工作区资源
- 选择适合当前任务的 Agent 工具

## 核心能力

| 能力 | 说明 |
|------|------|
| 平台概览 | AIASys 整体架构和定位 |
| Agent 工具目录 | 所有可用工具的说明和使用时机 |
| AutoTask 配合 | 连续推进任务的触发和停止信号 |
| UV 运行环境 | Python 依赖管理和环境绑定 |
| Docker 沙盒 | 隔离执行环境和资源限制 |
| 环境变量 | API key、token 等运行时配置 |
| 多维数据表 | 结构化数据的增删改查 |
| Skill 协作 | 多个 Skill 之间的调用关系 |
