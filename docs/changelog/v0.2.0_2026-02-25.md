# Changelog v0.2.0 - 2026-02-25

> 历史说明:
> 本文档记录的是当时版本的发布内容，其中 `.env.example`、认证模式等描述仅代表当时状态。
> 当前项目运行配置请以 `README.md` 与 `apps/backend/config.json` 为准。

## 概述

本次版本主要完成了认证系统重构（移除 SSO）、Docker 生产环境配置优化、以及部署流程标准化。

## 新增功能

### 认证系统
- 本地认证模式（Local Auth）完全替代 SSO
- JWT Token 认证机制
- 用户注册/登录/登出 API
- 数据库模型支持（SQLAlchemy + SQLite）

### Docker 生产环境
- 生产环境 Docker Compose 配置优化
- 自动环境检测（主机/Docker 内运行）
- IPython 沙盒容器自动适配路径
- Nginx 生产环境配置

### 部署流程
- 一键部署脚本（deploy.sh）
- 环境变量模板（.env.example）
- 服务器部署文档

## 修复问题

### 后端
- 修复 IPython 沙盒容器在 Docker 内启动失败
- 修复字体文件挂载路径问题
- 修复 Nginx 代理后端端口配置

### Docker
- 修复 Docker Compose 健康检查依赖
- 修复后端端口映射（8001 vs 18001）

## 优化改进

### 代码结构
- 移除 SSO 服务及所有相关代码
- 清理冗余部署脚本
- 统一 Docker 配置管理

### 文档
- 新增部署指南（DEPLOY.md）
- 更新 Docker README
- 更新 deploy README

## 破坏性变更

- **移除 SSO 认证**：所有 SSO 相关代码和配置已删除
- **环境变量变更**：`AUTH_MODE` 从 `sso` 改为 `local`
- **路径配置**：生产环境需要配置 `HOST_WORKSPACE_ROOT` 和 `HOST_FONT_PATH`

## 技术债务

- 后端容器内缺少 Docker CLI（不影响基本功能）
- 需要进一步优化 Docker 镜像大小
