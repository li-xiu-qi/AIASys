# Changelog v0.2.1 - 2026-02-26

## 概述

本次版本完成了运行环境功能的 UI 最终定稿和多项修复，将"执行空间"更名为"运行环境"，优化了弹窗布局、图标样式和数据加载逻辑。

## 新增功能

### 运行环境弹窗 (EnvironmentMarket)
- 弹窗式布局（非全屏，max-w-4xl h-[80vh]）
- 四个 Tab：全部 / 收藏 / 官方 / 自定义
- 搜索框（支持名称、描述、标签搜索）
- 分类筛选器（数据分析/机器学习/深度学习/科学计算/可视化/自定义）
- 来源筛选器（全部/官方/社区/自定义）
- 卡片网格布局（自适应宽度，minmax(320px, 1fr)）
- 统一灰色图标（BarChart3, Brain, Cpu, FlaskConical, LineChart, Settings）
- 创建环境按钮（搜索栏右侧）
- 选择环境按钮（卡片底部，防误触设计）

### 紧凑环境选择器 (CompactEnvSelector)
- 集成在数据分析页面 TopBar
- 下拉菜单显示当前环境
- 快速切换环境
- 快捷链接到环境市场

### 我的环境管理页面
- 独立页面 `/my-environments`
- 列出用户创建的自定义环境
- 显示构建状态（徽章）
- 查看构建日志功能
- 重新构建功能
- 删除环境功能

## 修复问题

### 界面修复
- 修复执行空间侧边栏点击无反应（添加 `onOpen` 回调）
- 修复环境弹窗双关闭按钮（隐藏 Dialog 默认按钮）
- 修复卡片高度不一致（auto-rows-fr + flex布局）
- 修复卡片误触问题（移除整体点击，添加独立按钮）
- 修复数据加载问题（弹窗打开时自动加载）

### 样式优化
- 统一图标颜色为 text-gray-600
- 统一图标尺寸 w-7 h-7
- 顶部装饰线改为 bg-gray-200
- 图标背景改为 bg-gray-50 rounded-lg
- 卡片底部按钮使用 outline 变体

## 命名统一

- "执行空间" → "运行环境"（统一命名）
- 侧边栏按钮：运行环境
- 弹窗标题：运行环境
- 页面标题：运行环境

## 移除功能

- 场景默认环境展示（保留后端数据）
- 卡片视图/列表视图切换（仅保留卡片视图）
- 彩色类别图标（统一为灰色）

## API 对接

| 功能 | API | 状态 |
|------|-----|------|
| 获取环境列表 | `GET /runtime-envs` | |
| 切换环境 | `POST /runtime-envs/sessions/{id}/switch` | |
| 收藏/取消 | `POST/DELETE /user/favorites/{id}` | |
| 设置默认 | `POST /user/default-env` | |
| 删除环境 | `DELETE /custom/{id}` | |
| 重新构建 | `POST /custom/{id}/rebuild` | |
| 构建状态 | `GET /custom/{id}/build-status` | |

## 技术实现

### 前端
- 使用 shadcn/ui Dialog 组件
- 使用 Tailwind CSS Grid auto-fill
- 使用 Lucide React 图标
- 自定义 Hook: useRuntimeEnvironments

### 后端
- Dockerfile 生成器安全校验
- IPythonBox 动态环境切换
- 多镜像提示词支持
