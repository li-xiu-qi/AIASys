# Changelog v0.4.26 - 2026-06-24

## li-xiu-qi

### 修复 (Bug Fixes)

#### 工作区文件上传同名覆盖

- 现象：工作区上传同名文件时，原文件被静默覆盖。
- 改动：上传接口检测到同名文件时，新文件自动追加递增后缀（-1、-2 ...），原文件保留。
- 文件变更
  - apps/backend/app/api/routes/workspaces_resources_files.py: 新增 _resolve_unique_file_path，工作区上传与全局工作区上传均应用唯一文件名逻辑
  - apps/backend/tests/test_workspace_file_routes.py: 补充同名上传冲突测试
