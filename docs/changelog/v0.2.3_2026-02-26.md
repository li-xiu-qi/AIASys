# Changelog v0.2.3 - 2026-02-26

## 概述

调整当前任务环境显示位置，从管理弹窗移动到聊天界面顶部，更清晰直观。

## 变更

### UI 调整
- 在 TopBar 下方添加当前任务环境显示条
- 显示环境名称和镜像信息
- 从管理弹窗中移除"当前任务使用环境"区域
- 管理弹窗只保留"默认环境"说明

### 显示位置对比

**之前**:
- 管理弹窗中显示"当前任务使用环境"
- 容易混淆（管理弹窗面向全部环境，不是当前任务）

**之后**:
- 聊天界面顶部显示"当前任务环境"
- 逻辑清晰：当前界面显示当前任务信息
