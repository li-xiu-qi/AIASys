# Changelog v0.2.7 (2026-03-05)

## 新增功能

### 双沙盒模式支持 (Docker / Local)
- 支持在创建任务时选择沙盒执行模式
- **Docker 沙盒**: 代码在隔离容器中执行，安全性高
- **本地沙盒**: 代码使用后端 Python 环境直接执行，启动更快
- 每个会话独立记录沙盒模式，同一会话内固定不变

### 本地代码执行工具 (LocalPythonTool)
- 实现本地受限 Python 代码执行
- 预装 pandas、numpy 等数据分析库
- 安全特性:
  - 禁用 os.system、subprocess、exec、eval 等危险函数
  - AST 静态代码检查
  - 执行超时控制（默认 30 秒，最大 120 秒）
  - 内存限制（512MB）

### 前端沙盒模式选择 UI
- 新建任务对话框添加沙盒模式选择
- 底部输入栏显示当前沙盒模式徽章（绿色=本地，蓝色=Docker）
- Local 模式时隐藏运行环境选择（仅 Docker 模式需要）
- 记住用户默认沙盒模式选择

## 架构改进

### 会话级沙盒模式配置
- SessionMetadata 添加 sandbox_mode 字段
- AgentService 根据会话模式动态加载对应配置
- 支持 Docker/Local 两种模式独立 Agent 配置

### 配置管理统一
- 删除 .env.example，统一使用 config.json 配置
- 配置简单明确，适合独立部署场景

## 文档更新

- 添加 ADR-004: 部署模式与配置策略架构决策记录
- 更新 SKILL.md 添加沙盒模式快速参考
- 详细说明沙盒模式与运行环境的区别

## 约束说明

**重要**: 沙盒模式选择后同一会话内不可更改
- Docker ↔ Local 之间不能切换
- Docker 模式内可切换不同运行环境（换镜像）
- 需要切换沙盒模式时，必须新建任务
