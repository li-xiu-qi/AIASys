# Changelog v0.4.28 - 2026-07-03

**更新日期**: 2026-07-03

## li-xiu-qi

### 新增功能 (Features)
- **桌面端版本号展示**:
  - 在左侧设计侧边栏顶部（品牌锁标右侧）增加当前版本号标签 `v0.4.28`。
  - 版本号读取自 `apps/web/package.json`，与桌面应用版本保持一致。

### 优化与改进 (Improvements)
- **文件预览安全头作用域扩展**:
  - `security_headers_middleware` 对工作区级 `/api/workspaces/{id}/files/download/` 和 `/api/workspaces/{id}/global-workspace/download/` 路径的 inline 预览移除 `X-Frame-Options` 等帧阻塞头，确保 PDF / 图片预览可正常嵌入。
  - 仅当 `disposition=inline` 时移除安全头，`attachment` 下载仍保留原有安全策略。
- **文件下载响应头简化**:
  - 文件下载接口改用 Starlette `FileResponse` 的 `filename` + `content_disposition_type="inline"` 参数，替代手动拼接 `Content-Disposition`，避免特殊字符转义不一致。

### 修复 (Bug Fixes)
- **Canvas / 工作区文件预览 URL 解析**:
  - `createWorkspacePreviewFile` 增加 `workspaceId` 参数，优先按工作区作用域生成预览和下载 URL。
  - 修复 Canvas 拖拽打开文件、右侧工作区资源面板预览等场景下 URL 仍绑定 `sessionId` 导致会话切换后链接失效的问题。

### 文档完善 (Documentation)
- 新增本版本变更记录 `docs/changelog/v0.4.28_2026-07-03.md`。

### 测试 (Tests)
- `apps/backend/tests/test_file_preview_security_headers.py`:
  - 增加工作区级 inline / attachment 预览安全头回归测试。
  - 增加全局工作区 inline 预览安全头回归测试。
