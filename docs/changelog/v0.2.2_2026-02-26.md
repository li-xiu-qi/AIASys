# Changelog v0.2.2 - 2026-02-26

## 概述

本次版本完成了 Docker 容器监控和管理功能，用户现在可以查看所有正在运行的 Docker 容器，并进行停止操作。

## 新增功能

### Docker 容器监控面板
- 显示用户所有运行中的 Docker 容器
- 显示容器信息：ID、名称、环境类型、绑定会话
- 显示运行时长、健康状态
- 自动刷新（每10秒）
- 手动刷新按钮

### 容器管理功能
- 停止指定容器
- 停止前确认对话框
- 当前会话容器特殊标记（蓝色标签）
- 停止当前会话时警告提示

### API 接口
- `GET /api/runtime-envs/containers` - 获取运行中容器列表
- `POST /api/runtime-envs/containers/{pool_key}/stop` - 停止容器

## UI 更新

### 运行环境管理弹窗
- 新增 "运行中" Tab（仅管理模式显示）
- 显示运行中容器数量徽章
- 空状态时显示引导提示

### 容器卡片信息
- 容器短ID（12位）
- 环境名称和分类
- 绑定会话ID
- 运行时长（自动计算）
- 健康状态指示器
- 停止按钮（带确认）

## 技术实现

### 后端
- `IPythonBox.list_active_containers()` - 获取容器列表
- `IPythonBox.stop_container_by_key()` - 停止指定容器
- 新增 `RunningContainerInfo` 模型

### 前端
- `RunningContainersPanel` 组件
- `useRuntimeEnvironments` 添加容器监控方法
- 自动刷新机制

## 安全考虑
- 只能停止自己的容器（通过 workspace_path 前缀验证）
- 停止前需要确认
- 当前会话容器有额外警告

## 待优化

- 容器资源使用显示（CPU/内存）
- 批量停止功能
- 容器日志查看
