# 知识管理工作区记忆

## 长期目标
- 建立可持续积累的个人/团队知识库
- 实现从原始资料到结构化知识的自动化转化
- 让知识可以被检索、关联和复用

## 关键术语
- LLM Wiki：由 LLM 维护的 Markdown 知识 wiki，知识在摄入时编译而非查询时重新推导
- Raw Sources：原始资料层，论文、网页、笔记等，只读不修改
- Wiki Layer：知识合成层，概念页面、交叉引用、[[wikilinks]] 链接
- 知识库（KB）：AIASys 向量知识库，支持文档上传和语义检索
- 知识图谱（KG）：AIASys 图谱系统，自动抽取实体和关系，支持社区发现
- Ingest Pipeline：资料摄入 → 路由 → 合成 → 嵌入 → 更新索引
