# Changelog v0.2.4 (2026-02-27)

## 重要修复

### Docker Workspace 权限问题
- **问题**: Docker 容器无法写入 workspace，导致保存图片失败
- **错误**: `Permission denied: '/workspace/kmeans.png'`
- **根因**: Docker 安全选项 (`--security-opt no-new-privileges` + `--cap-drop ALL`) 与宿主机目录权限冲突
- **修复**:
  - `agent_service.py`: 创建目录时自动设置 777 权限
  - `ipython_tool.py`: 容器启动前设置 workspace 目录权限
