# Changelog v0.1.0 - 2026-05-28

首个对外发布版本。新仓库初始化，代码从内部迭代仓库迁移。

## 核心架构

### Agent 执行引擎
- Agent 执行与会话管理：支持流式 SSE 输出、会话分支、执行控制
- Agent 配置：提示词模板、工具列表、默认模型选择
- 会话预算管理：token 预算设置、消耗追踪、上下文占用计算
- 会话生命周期：创建、删除、导出、监控

### 工作区系统
- 工作区 CRUD 与模板创建
- 全局工作区：跨工作区共享资源与配置
- 文件管理：文件树浏览、创建、编辑、差异对比
- 资源挂载与验活

### Notebook 工作台
- 多 Cell 类型支持（代码、Markdown、原始文本）
- Cell 执行引擎与输出渲染（文本、HTML、图片、JSON、SVG、流式）
- Notebook 检查器：差异对比、大纲、执行历史、变量查看

### 多维数据表
- 数据表 CRUD，支持行列操作与类型推断
- 全局数据表支持

### 知识库
- 向量检索：文档上传、分块、嵌入、语义搜索
- 知识图谱（GraphRAG）：实体提取、消歧、社区检测、图查询
- PixiJS 画布可视化

### 数据库连接
- 外部数据库连接器管理（PostgreSQL、MySQL、SQLite 等）
- 运行时数据库查询（DuckDB）
- 会话级数据库附件
- 工作区 SQLite/DuckDB 文件直接查询

### AutoTask 自动化
- 四种触发类型：连续执行、定时周期、Cron 表达式、单次执行
- 任务调度引擎与执行策略
- 执行结果记录与状态追踪

### Memory 子系统
- 两层作用域：用户全局 + 工作区层
- 两阶段 Pipeline：提炼记忆 → 写入 Markdown
- 安全扫描：不可见 Unicode、提示注入、凭证泄露、持久化攻击检测
- 版本管理与快照哈希

### 统一能力管理
- 三种能力类型：Skill Pack、MCP Server、Subagent
- 能力安装、卸载、激活、验活
- 内置 12 个系统 Skill：Canvas 编辑、PDF 翻译、OCR、论文搜索、竞赛研究、Memory 整理等
- Skill 市场与 MCP 市场入口

### Claw 通信通道
- 微信/IM 连接绑定
- 消息收发、附件导入、远程确认

### 桌面应用
- Electron 41 薄壳，复用 Web 前端
- Linux 目录版打包验证通过
- 开发态复用 Vite dev server，预览态复用构建产物
- packaged 模式 backend 运行态外置到用户目录

## 前端界面

### 页面
- 首页：产品介绍与能力展示
- 主工作区（`/workspace`）：三栏布局（ActivityBar + 主画布 + 对话侧栏）
- 知识图谱可视化页（`/knowledge/graph`）：PixiJS 交互画布
- 用户设置页（`/profile`）：账户安全、LLM 配置、系统信息

### ActivityBar 导航（12 项）
当前工作区、全局工作区、数据查询、文件搜索、监控任务、终端、自动化任务、专家协作节点、环境变量、能力管理、工作区设置、文件变更

### 核心交互
- 对话输入区与流式消息渲染
- 文件预览面板（文本、图片、Canvas、数据表）
- Notebook 编辑器与 Cell 执行
- 知识库上传与检索
- 数据库查询结果浏览
- 终端（PTY）集成

## 基础设施

- 后端：FastAPI + SQLAlchemy + DuckDB + SQLite
- 前端：React 19 + Vite + Tailwind CSS 4 + shadcn/ui
- 桌面：Electron 41 + electron-builder
- 测试：Pytest（后端）+ Playwright（前端 E2E）
- 代码质量：Ruff + ESLint + Lefthook pre-commit hooks
- 三端支持目标：Windows / macOS / Linux

## 已知限制

- 桌面版 Windows 和 macOS 打包尚未完成真实验证
- 前端不做手机移动端适配（<768px 视口）
- 多用户协作暂未支持，当前按单机单用户设计
